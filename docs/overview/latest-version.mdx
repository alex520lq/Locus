## 变更列表

### 新增

- 新增 `unity_yaml_list` 和 `unity_yaml_search`，用于在读取对象详情前检查 Unity 场景与 Prefab 层级。
- `unity_yaml_read` 新增更细的详情模式，支持实时 Editor 读取、Prefab override、序列化字段和重复名称安全路径。
- Chat 会话渲染新增隐藏已完成 thinking 的显示设置，并按真实顺序展示 thinking、工具调用和流式文本。
- Markdown 与 Chat 资产引用支持更多 Unity、工作区和 `ProjectSettings` 路径格式。

### 修复

- 修复自定义模型端点的推理参数配置，OpenAI Chat、OpenAI Responses、Anthropic 兼容端点现在可以发送正确的推理参数格式，并显示可用推理强度。
- 启动和切换工作区时会校准已加载的资产数据库，处理过期引用图、同路径 GUID 冲突、大小/哈希变化和脚本级联更新。
- Unity 域重载期间保留嵌入式 WebView，并对临时关闭执行延迟清理。
- Knowledge lexical rebuild 进度改为接收状态事件推送。
- Settings 会缓存调试模式状态，重新挂载设置面板时保持开关状态稳定。
- Chat 文件变更撤销执行中会锁定撤销按钮，避免重复触发。
- 改进 Subagent 和 Unity recompile 工具调用的等待状态展示。
