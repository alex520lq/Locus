## Changes

### Added

- Added `unity_yaml_list` and `unity_yaml_search` for inspecting Unity scene and prefab hierarchies before reading object detail.
- Added richer `unity_yaml_read` detail modes for live Editor reads, prefab overrides, serialized fields, and duplicate-safe object paths.
- Added chat transcript controls for hiding completed thinking blocks and preserving the actual order of thinking, tool calls, and streamed text.
- Added broader Markdown and chat asset reference handling for Unity, workspace, and `ProjectSettings` paths.

### Fixed

- Fixed custom endpoint reasoning configuration so OpenAI Chat, OpenAI Responses, and Anthropic-compatible endpoints can send the correct reasoning parameter format and expose supported effort levels.
- Reconciled loaded asset databases on startup and workspace switch so stale reference graph rows, same-path GUID conflicts, size/hash changes, and script cascades are detected.
- Kept the Unity embedded WebView alive during Unity domain reloads, with delayed cleanup for transient reload closes.
- Updated Knowledge lexical rebuild progress to use pushed status events instead of polling.
- Cached debug mode state in Settings so remounted settings panels keep the correct toggle state.
- Locked chat file-change undo while an undo operation is already running.
- Improved subagent and Unity recompile tool-call waiting states.
